# klasifikasi-makanan-minangkabau-mobilenetV3
Proyek ini mengimplementasikan model MobileNetV3 untuk melakukan klasifikasi citra makanan khas Minangkabau ke dalam sembilan kelas, yaitu: telur_dadar, telur_balado, gulai_tunjang, gulai_tambusu, gulai_ikan, dendeng_batokok, daging_rendang, ayam_pop, dan ayam_goreng berbasis citra digital dan dilengkapi dengan deployment model melalui Fast API
