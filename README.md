# klasifikasi-makanan-minangkabau-mobilenetV3
Proyek ini mengimplementasikan model mobilenetV3 untuk mengklasifikasikan makanan minangkabau dengan 9 class yaitu: telur_dadar,telur_balado,gulai_tunjang,gulai_tambusu,gulai_ikan,dendeng_batokok,daging_rendang,ayam_pop,ayam_goreng berbasis citra digital serta dilengkapi dengan integrasi ke aplikasi berbasis mobile (flutter)
